# HDIC project site

Hugo で構築した研究サイトのソースリポジトリです。

- 公開サイト: https://shikeda.github.io/
- テーマ: [hugo-book](https://github.com/alex-shpak/hugo-book)
- デプロイ: GitHub Actions による自動ビルド

## ローカル環境での実行

​```bash
git clone --recurse-submodules https://github.com/shikeda/shikeda.github.io.git
cd shikeda.github.io
hugo server
​```
