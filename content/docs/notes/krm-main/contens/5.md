# krm_pronunciations


名義抄の音注は、反切、類音注、仮名注があり、それらに声点が施されることも多い。
日本漢字音のデータベースとしては、加藤大鶴氏らによる
「資料横断的な漢字音・漢語音データベース」（略称DHSJR）が非常に充実した
内容を持っている。またその仕様も詳細に公開されている。
DHSJRの仕様に合わせたデータ公開を検討中である。
