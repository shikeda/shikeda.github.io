# krm_ndl

国会図書館デジタルコレクションで公開している
観智院本類聚名義抄の画像へのリンクを整理したものである。
観智院本類聚名義抄の所在と国会図書館デジタルコレクションのURLとを対照させたデータである。
ファイル名は、KRM_ndl.tsvである。

KRM_ndl.tsvは2025年3月の仕様変更の影響がほとんどないので
説明は省略する。
