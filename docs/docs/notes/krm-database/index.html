    類聚名義抄 #

    『類聚名義抄』は日本古辞書の雄編として名高い。 書名の読み方は「るいじゅみょうぎしょう」である。 源順撰『倭名類聚抄』の「類聚」と、空海撰『篆隷万象名義』の 「名義」とを採用して書名としたとされる。

    ここでは、まず『類聚名義抄』とその諸本の概要を述べる。 次に『類聚名義抄』の完本として唯一の伝本である 観智院本について、その項目の構造を解説し、 その後、観智院本の本文入力方法の詳細を述べる。
